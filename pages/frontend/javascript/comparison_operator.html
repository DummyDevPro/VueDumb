is equal operator (==)

* Make comparison between two components(participants) which are equal?
* If the two variable are equal, return true otherwise false

TRUE example let numOne = 10;
let numTwo = 10;
let isEqual = (numOne == numTwo);

// Let see the output
console.log(isEqual); // the output will be true
