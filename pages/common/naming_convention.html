Naming Convention

Naming Convention Format Language Explain Examples
Kebab Case keba-case HTML & CSS Use small letters.
When more than one word, concats '-' between words.
  • id-name
  • class-name
  • element-name-property
  • etc...
Camel Case camelCase JavaScript Use small letter for one word.
When more than one word,
first word is small letter
but the other word's first letter should be capital letter.
  • variableName
  • functionName
  • etc...

HTML

Elements

  • HTML element's name are not case-sensitive.
  • But should use lowercase letters.

Element's attributes name

  • Names are case-sensitive, lowercase and uppercase are different.
  • Names should use keba case.
  • Names should be meaningful.

JavaScript

Variable

  • Names are case-sensitive, lowercase and uppercase are different.
  • Start variable names with a letter and should use camel case.
  • Variable names should be meaningful.

Functions

  • Names are case-sensitive, lowercase and uppercase are different.
  • Start function names with a letter and should use camel case.
  • Function names should be meaningful.
